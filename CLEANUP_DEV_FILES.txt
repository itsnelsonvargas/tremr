TREMR - Development Files Cleanup Guide
================================================================================

The following files are development/testing files that can be safely deleted
if you're preparing for final distribution. They are already in .gitignore.

SAFE TO DELETE (Development/Testing):
================================================================================

1. rename_to_tremr.py
   Purpose: Script used to rename project from old name to Tremr
   Status: Task completed, no longer needed
   Action: DELETE

2. complete_rename_tremr.py
   Purpose: Comprehensive rename script
   Status: Task completed, no longer needed
   Action: DELETE

3. test_phivolcs_html.py
   Purpose: Test script for analyzing PHIVOLCS HTML structure
   Status: Development tool, not needed in production
   Action: DELETE

4. gui_old.py
   Purpose: Old backup of GUI file
   Status: Backup file, not used
   Action: DELETE

5. system_test.log
   Purpose: Log file from system tests
   Status: Generated file, will be recreated if tests run again
   Action: DELETE


OPTIONAL TO KEEP (Testing/Verification):
================================================================================

1. test_monitor.py
   Purpose: Test with mock earthquake data
   Status: Useful for testing without internet
   Action: KEEP (included in distribution)

2. test_full_system.py
   Purpose: Comprehensive system test suite
   Status: Useful for verifying installation
   Action: KEEP (included in distribution)

3. test_live_phivolcs.py
   Purpose: Test with real PHIVOLCS data
   Status: Useful for verification
   Action: KEEP (but not in distribution)


GENERATED FILES (Auto-created):
================================================================================

These files are generated automatically and ignored by git:

1. earthquake_monitor.log
   - Application log file
   - Auto-created when app runs

2. seen_earthquakes.json
   - Tracks seen earthquakes
   - Auto-created when app runs

3. system_test_report_*.json
   - Test result reports
   - Generated by test_full_system.py

4. earthquake_warning.ico
   - Warning icon
   - Auto-created by application

5. dist/ folder
   - Distribution packages
   - Generated by create_distribution.py


QUICK CLEANUP COMMAND:
================================================================================

To delete all development files, run:

Windows:
  del rename_to_tremr.py complete_rename_tremr.py test_phivolcs_html.py gui_old.py system_test.log 2>nul

Linux/Mac:
  rm -f rename_to_tremr.py complete_rename_tremr.py test_phivolcs_html.py gui_old.py system_test.log
